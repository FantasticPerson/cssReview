```
使用百分数设置margin，规范的作者没有简单的忽略上下外边距，而是决定让它与父元素的width相关，不会根据其后代元素的width而改变
```

```
设置外边距时必须小心。你很可能想抹去标题和下一段之间的空间。你可以使用兄弟原则做到这一点
h2{
    margin-bottom:0;
}
h2 + p{
    margin-top:0;
}

或者
h2{
    margin-bottom:0;
}
p{
    margin:0 0 1em;
}
这会除掉段落的上外边距，不过由于段落还有一个1em的下外边距，所以还会保留这段段间间隙
```

```
外边距还可以应用到行内元素，不过不影响行高，所以没有背景的情况下，margin-top 和 margin-bottom 没有任何效果

左右外边距还是可见的
```

```
 行内非替换元素和行内替换元素背景的边界之间有额外的一个空间。如果愿意，可以在行内元素的两端都增加这个额外的空间
```

```
行内非替换元素上下的边框不影响 line-height
```
